## 🧱 REGLAS TÉCNICAS OBLIGATORIAS

### 1. **WordPress Best Practices**
```php
// ✅ BIEN
add_action('wp_enqueue_scripts', 'ucondieresis_enqueue_styles');
function ucondieresis_enqueue_styles() {
    wp_enqueue_style('ucondieresis-style', get_template_directory_uri() . '/assets/css/main.css');
}

// ❌ MAL
<link rel="stylesheet" href="/wp-content/themes/ucondieresis/style.css">
```

### 2. **URLs Dinámicas - NUNCA Hardcodear**
```php
// ✅ BIEN
$home = home_url('/');
$template_uri = get_template_directory_uri();
$image = $template_uri . '/assets/images/logo.svg';

// ❌ MAL
$image = '/wp-content/themes/ucondieresis/assets/images/logo.svg';
```

### 3. **Escapar Datos Dinámicos**
```php
// ✅ BIEN - Escapar según contexto
echo esc_html($post_title);              // HTML
echo esc_url($link);                     // URLs
echo wp_kses_post($content);             // HTML con tags permitidos
echo esc_attr($data_attribute);          // Atributos

// ❌ MAL
echo $post_title;
```

### 4. **Estructura HTML Semántica**
```html
<!-- ✅ BIEN -->
<article class="post" id="post-<?php the_ID(); ?>">
    <header class="post-header">
        <h1 class="post-title"><?php the_title(); ?></h1>
    </header>
    <section class="post-content">
        <?php the_content(); ?>
    </section>
    <footer class="post-footer">
        <?php wp_link_pages(); ?>
    </footer>
</article>

<!-- ❌ MAL -->
<div id="contenido">
    <div class="titulo"><?php the_title(); ?></div>
    <div class="texto"><?php the_content(); ?></div>
</div>
```

### 5. **CSS: No Estilos Inline**
```php
// ✅ BIEN - En style.css o archivo externo
<div class="hero-section">
    <h1 class="hero-title"><?php echo esc_html($title); ?></h1>
</div>

// ❌ MAL
<div style="background: url('...'); padding: 50px;">
    <h1 style="color: #333; font-size: 48px;"><?php echo $title; ?></h1>
</div>
```

### 6. **Prefijo de Funciones: `ucondieresis_`**
```php
// ✅ BIEN
function ucondieresis_get_hero_content() {
    return apply_filters('ucondieresis_hero_content', []);
}

function ucondieresis_render_testimonial($testimonial) {
    // Lógica...
}

// ❌ MAL
function get_hero_content() { ... }
function render_testimonial() { ... }
```

### 7. **Evitar Contaminación del Scope Global**
```php
// ✅ BIEN - Usar scoping
class Ucondieresis_Theme {
    public function init() {
        add_action('wp_enqueue_scripts', [$this, 'enqueue_scripts']);
    }
    
    public function enqueue_scripts() {
        // ...
    }
}

// ❌ MAL
global $ucondieresis_config;
$ucondieresis_config = [...];
```

### 8. **Hooks WordPress Correctamente**
```php
// ✅ BIEN - Usar hooks existentes
add_action('wp_footer', 'ucondieresis_add_footer_scripts');

// ✅ BIEN - Permitir que otros plugins filtren
$content = apply_filters('ucondieresis_page_content', $content);

// ❌ MAL - No buscar "puntos de extensión"
echo $content; // Sin posibilidad de filtro
```

### 9. **PHP 8+ Compatible**
```php
// ✅ BIEN - Tipos y null-safe
function ucondieresis_get_option(string $key): ?string {
    return get_option($key);
}

// ✅ BIEN - Match sobre switch
$type = match($post_type) {
    'post' => 'Artículo',
    'page' => 'Página',
    default => 'Contenido'
};

// ❌ EVITAR - Código PHP 7.0
```

## 📦 SI ES TEMPLATE WORDPRESS

Debe incluir siempre:

```php

/**
 * Template: {Nombre}
 * Description: {Descripción}
 * 
 * @package Ucondieresis
 */

get_header();
?>

<main id="primary" class="site-main">
    <!-- Contenido -->
</main>

<?php get_footer();
```

### Requisitos
- ✅ Abrir con `get_header()`
- ✅ Cerrar con `get_footer()`
- ✅ Usar WordPress Loop si aplica
- ✅ Permitir edición futura (ACF/Gutenberg)
- ✅ Preparado para i18n: `__('Texto', 'ucondieresis')`
- ✅ Documentación en bloque PHP
