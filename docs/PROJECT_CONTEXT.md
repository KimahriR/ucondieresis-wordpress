# Project Context

## Overview

U con Diéresis is a custom WordPress project designed as a premium visual showroom.

The site is focused on:
- Showcasing personalized products
- Offering downloadable PDF catalogs
- Driving direct conversation through WhatsApp

This is not a traditional e-commerce website.

## Core Business Model

Primary goals:
- Showcase personalized products
- Offer downloadable PDF catalogs
- Drive direct WhatsApp inquiries

This project should avoid e-commerce assumptions such as:
- Shopping cart flows
- Checkout logic
- Public pricing systems
- Complex inventory logic

Primary conversion channel:
- WhatsApp

## Brand Positioning

The brand should feel:
- Premium
- Warm
- Elegant
- Handcrafted
- Personal

Avoid:
- Generic corporate language
- Aggressive sales copy
- Marketplace or store-like UX patterns

## Main Content Areas

- Homepage showroom
- Product archives and product detail pages
- Downloadable catalogs
- Inspiration and brand presentation
- Contact and WhatsApp conversion flows

## User Journey

The expected user flow is simple:
- Discover the brand
- Explore products or inspiration
- Download a catalog or review featured items
- Start a WhatsApp conversation

The site should reduce friction and move users naturally toward contact.

## Project Direction

The project should continue evolving as:
- A modular custom WordPress build
- A premium brand experience
- A lightweight and maintainable codebase
- A conversion-focused showroom, not a store

## Documentation Philosophy

Project documentation should stay:
- Clear
- Lightweight
- Useful in daily work
- Easy to update as the project evolves

These docs are intended to evolve with the codebase, not to remain static.
